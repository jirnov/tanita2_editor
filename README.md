# Trickster games editor (C++ part)

## Для сборки необходимо установить:
* [DirectX 9 SDK](https://download.microsoft.com/download/a/e/7/ae743f1f-632b-4809-87a9-aa1bb3458e31/DXSDK_Jun10.exe)
* [Visual Studio 2008 Express](https://download.microsoft.com/download/E/8/E/E8EEB394-7F42-4963-A2D8-29559B738298/VS2008ExpressWithSP1ENUX1504728.iso)
* [CMake 3.25.3](https://cmake.org/files/v3.25/cmake-3.25.3-windows-i386.msi)
* Windows 7 

## Подготовка к сборке:
```cmd
mdkir build
cd build
cmake -A Win32 ..
```

## Сборка:
```cmd
cd build
cmake --build .
```
